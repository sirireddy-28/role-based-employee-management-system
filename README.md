Employee Role Management System

The Employee Role Management System is a comprehensive web application developed to simplify organizational workforce management. It enables administrators to manage employee records, assign roles and permissions, monitor attendance, track performance metrics, and maintain a structured workflow through a centralized dashboard.

Key Features
Secure User Authentication and Authorization
Role-Based Access Control (Admin, Manager, Employee)
Employee Profile Management
Attendance Tracking and Monitoring
Performance Evaluation and Reporting
User-Friendly Dashboard Interface
Database Integration for Data Storage
Responsive and Easy-to-Use Design
Technologies Used
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Database: SQLite
Version Control: Git & GitHub
Project Objective

The primary objective of this project is to provide an efficient, secure, and scalable solution for managing employee information, organizational roles, attendance records, and performance evaluations while reducing manual administrative effort.
